# Playstation VR tutorial #

### What is this repository for?  
To use PSVR on web

![line](http://i.imgur.com/aJfFBV7.png)
### How do I get set up?
```sh
$ git clone https://github.com/cdra0624/psvr-tutorial.git
$ cd psvr-tutorial
$ npm install
$ npm start
```  
localhost:7788 is index page so you can go each page from there.

![line](http://i.imgur.com/aJfFBV7.png)
